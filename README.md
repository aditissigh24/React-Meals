# React-Meals
A modern restaurant website built with React.  
This site showcases the restaurant’s menu, provides prices for each dish, and allows users to add dishes to their cart for an easy ordering experience.  
## Features
-Menu Display: Browse through the restaurant's menu with detailed dish descriptions and prices.  
-Cart Functionality: Add dishes to the cart for review and checkout.  
-Real-time Updates: View updated cart totals and item quantities.  
## Technologies Used
-React: A JavaScript library for building user interfaces.  
-CSS: For styling the application.
